1. Open the project folder and run this path in cmd.
2. Run command pip install -r requirements.txt
3. After the installations completed run command python manage.py runserver.
4. You will get an ip address http://127.0.0.1:8000/. execute this ip address in your browser
5. Now you have to create an superuser for that in cmd write command python manage.py createsuperuser. give username, email address and password.
